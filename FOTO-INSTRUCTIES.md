## üìã STAP-VOOR-STAP HANDLEIDING

### Methode 1: Via Browser (Eenvoudigst)

1. **Open huidige website:**
   ```
   https://mondzorgsloterweg.nl/
   ```

2. **Per foto:**
   - Rechtsklik op de foto
   - Kies "Bewaar afbeelding als..."
   - Sla op in Downloads

3. **Verplaats naar nieuwe website:**
   - Open Finder
   - Ga naar Downloads
   - Selecteer de foto's
   - Sleep naar de juiste map:
     * Team foto's ‚Üí `assets/team/`
     * Hero foto's ‚Üí `assets/`
     * Iconen ‚Üí `assets/icons/`

4. **Hernoem:**
   - Rechtsklik op bestand
   - Kies "Wijzig naam"
   - Typ nieuwe naam (zie hierboven)

---

### Methode 2: Via Website Inspector (Geavanceerd)

1. **Open huidige website**
2. **Rechtsklik ‚Üí Inspecteer**
3. **Ga naar Network tab**
4. **Refresh pagina**
5. **Filter op "Img"**
6. **Zie alle afbeeldingen URLs**
7. **Kopieer URL en download**

---

### Methode 3: Via cPanel/Hosting (Beste Kwaliteit)

Als u toegang heeft tot uw hosting:

1. **Log in op hosting (cPanel/FTP)**
2. **Ga naar:**
   ```
   public_html/wp-content/uploads/
   ```
3. **Download hele uploads map**
4. **Zoek de foto's**
5. **Kopieer naar nieuwe website**
